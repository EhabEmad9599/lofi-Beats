# Lfoi-beats

# Installing
to get the project running, follow these steps:

- clone this repository:

```html
https://github.com/EhabEmad9599/lofi-Beats.git
```
- install all the project's dependencies
``` html
npm install
```
- once  that's done, run the project:

```html
npm start
```
